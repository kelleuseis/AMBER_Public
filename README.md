# AMBER_Public
AMBER contains labelled waveforms from microseismic events recorded on deep downhole sensor arrays. It has been generated to stimulate research and development in the 
use of AI tools for downhole microseismic processing tasks. For more information, please see Leung et al. (2026, in prep).
